# Module: creative
## Purpose
Creative reasoning patterns for ideation, design, lateral thinking, and aesthetic decisions.

## Activation Signals
- "design", "create", "ideate", "brainstorm", "concept", "theme", "palette", "layout"
- "logo", "brand", "ui", "ux", "visual", "animation", "effect", "style"
- "imagine", "what if", "play with", "experiment", "try", "vibe"

## Cognitive Patterns
1. **Diverge then converge** — Generate many ideas first, then select and refine.
2. **Constraint as fuel** — Limitations breed creativity. Embrace constraints rather than fighting them.
3. **Reference harvesting** — Look at real-world examples before designing. Deconstruct what works.
4. **Aesthetic coherence** — Every element should serve the whole. Consistency > novelty.
5. **User emotion** — Design for feeling, not just function. How should this make Andre feel?
6. **Rapid iteration** — Sketch broadly, refine narrowly. Don't polish the first draft.

## Inhibitions
- Never start with the most obvious idea — explore first.
- Never sacrifice usability for aesthetics.
- Never ignore the existing design system unless explicitly asked to break it.
